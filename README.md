# 🛒 Shopping Cart Backend

[![Java](https://img.shields.io/badge/Java-100%25-blue?logo=java)](https://www.java.com/)
[![Spring Boot](https://img.shields.io/badge/Spring%20Boot-2.7%2B-brightgreen?logo=spring)](https://spring.io/projects/spring-boot)

A robust e-commerce web application backend built with Spring Boot, handling core shopping cart functionality. **Note:** This project is intentionally built without Spring Security to allow for flexible integration. Users are expected to add their own security configuration.

## 🚧 Project Status

This is an initial commit version. The core structure and dependencies are in place, but the project is in active development.

## ✨ Features

- **E-commerce Core Logic:** Backend services for managing a shopping cart.
- **Spring Boot Foundation:** Built on the reliable Spring Boot framework.
- **Ready for Security:** Designed to easily integrate Spring Security or your preferred authentication method.
- **Maven Managed:** Uses Maven for dependency management and build automation.
- **Production Ready Setup:** Includes Maven Wrapper (`mvnw`) for consistent builds.

## 📋 Prerequisites

Before you begin, ensure you have the following installed:
- **Java Development Kit (JDK):** Version 11 or higher (recommended).
- **Maven:** (Optional, as the project includes the Maven Wrapper).
